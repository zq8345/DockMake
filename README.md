# DockMake

DockMake corporate website.

This is a static site for `dockmake.com`, designed as the parent company/brand presence rather than a product-specific page.

## Deploy

Deploy the repository as a static site:

- Build command: none
- Output directory: `/`
- Entry point: `index.html`

## Local preview

Open `index.html` directly in a browser, or serve the folder with any static server.
